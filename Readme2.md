## Қазақша нұсқа

### 1. Жобаның мақсаты
RoboLearn - робототехника бойынша оқу процесін цифрландыруға арналған full-stack веб-портал.

Негізгі мақсаттар:
- мұғалімге тапсырма беру және бағалау процесін жеңілдету;
- студентке жобаны ыңғайлы түрде тапсыру;
- AI арқылы бастапқы кері байланыс ұсыну;
- оқу аналитикасын бір жерден көру.

### 2. Қолданушылар рөлі
- **Teacher (мұғалім):** тапсырма құрады, студенттер тізімін қарайды, бағалайды, аналитиканы көреді.
- **Student (студент):** тапсырма алады, жоба жібереді, пікір/баға көреді.

### 3. Технологиялық стек
#### Frontend
- React 18
- Vite
- TypeScript
- Tailwind CSS
- React Router
- Axios
- i18next (қазақша/орысша/ағылшынша)

#### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Multer
- Zod

#### AI
- Gemini API (қосулы болса)
- fallback логика

### 4. Архитектура
Жоба қабатталған архитектурамен жасалған:
- `routes` - маршруттар
- `controllers` - бизнес-логика
- `models` - MongoDB схемалары
- `middlewares` - auth/role/error
- `utils` және `services` - көмекші және интеграциялық логика

Frontend-та:
- `pages` - бет логикасы
- `components` - қайта қолданылатын UI
- `contexts` - глобалды күй (мысалы, theme)
- `lib` - API клиент, download helper

### 5. Қауіпсіздік
- JWT арқылы авторизация
- рөлге негізделген рұқсат (RBAC)
- teacher-only endpoint-тер
- валидация (Zod)
- файл атауларын қауіпсіз өңдеу
- UUID арқылы файлдарды бірегей сақтау (overwrite болдырмау)

### 6. Негізгі модульдер
#### Тапсырма модулі
- тапсырма құру/жаңарту/архивтеу/қалпына келтіру/жою
- мұғалім файл тіркей алады

#### Жоба тапсыру модулі
- студент атау, сипаттама, құрал таңдап жоба жібереді
- файл жүктеу (документ және видео форматтар)

#### Бағалау модулі
- AI бастапқы бағалау
- мұғалімнің финал бағасы

#### Пікірлер модулі
- project ішінде комментарий алмасу

#### Студенттер модулі
- студенттер тізімі
- фильтр/экспорт
- студентті жою (каскадты тазалау)

#### Аналитика модулі
- орташа балл, орындалу пайызы, top students

### 7. Theme және UX
- Global dark/light theme
- localStorage-та сақтау
- барлық бетте бірізді дизайн

### 8. Көптілділік
Жүйе 3 тілді қолдайды:
- Қазақша
- Орысша
- Ағылшынша

### 9. Деректер тұтастығы
Студент жойылған кезде оған байланысты деректер де жойылады:
- `Project`
- `Rating`
- `Comment`
- байланысты файлдар

Бұл orphan data қалдырмау үшін керек.

### 10. Өнімділік және тұрақтылық
- `async/await` + loading states
- қате өңдеу (UI сынбайды)
- build және lint тексерістері
- production-ready API жауап форматы

### 11. Диплом қорғауға қысқа мәтін
> Менің дипломдық жобам - RoboLearn веб-порталы.
> Бұл жүйе мұғалім мен студент арасындағы оқу циклін толық автоматтандырады:
> мұғалім тапсырма береді, студент жоба тапсырады, AI бастапқы бағалау ұсынады,
> мұғалім қорытынды бағасын қояды.
> Жоба React + Node.js + MongoDB технологияларымен жасалды.
> Қауіпсіздік JWT және рөлдік авторизация арқылы шешілді.
> Көптілділік, dark/light режимі, файл жүктеу, аналитика толық енгізілді.
> Нәтижесінде оқу процесіне дайын, тұрақты және масштабталатын платформа жасалды.

## Русская версия

### 1. Цель проекта
**RoboLearn** - full-stack веб-портал для цифровизации учебного процесса по робототехнике.

Ключевые задачи:
- упростить работу преподавателя с заданиями и проверкой;
- дать студенту удобную отправку проекта;
- добавить первичную AI-оценку;
- предоставить аналитику по обучению.

### 2. Роли пользователей
- **Teacher:** создание заданий, проверка, аналитика, работа со студентами.
- **Student:** просмотр заданий, отправка проекта, получение обратной связи.

### 3. Технологии
#### Frontend
- React 18
- Vite
- TypeScript
- Tailwind CSS
- React Router
- Axios
- i18next (KZ/RU/EN)

#### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Multer
- Zod

#### AI
- Gemini API (при наличии ключа)
- fallback логика

### 4. Архитектура
Применена layered-архитектура:
- `routes`
- `controllers`
- `models`
- `middlewares`
- `utils/services`

На frontend:
- `pages`
- `components`
- `contexts`
- `lib`

### 5. Безопасность
- JWT-аутентификация
- RBAC (ограничение по ролям)
- teacher-only endpoints
- строгая валидация данных
- безопасное именование файлов
- UUID в имени файла для предотвращения перезаписи

### 6. Основной функционал
#### Модуль заданий
- создание/обновление/архивация/восстановление/удаление заданий
- прикрепление материалов

#### Модуль отправки проекта
- студент отправляет описание + файл
- поддержка документов и видео

#### Модуль оценивания
- AI-оценка
- ручная оценка преподавателя

#### Комментарии
- обсуждение внутри проекта

#### Модуль студентов
- список студентов
- фильтрация/экспорт
- удаление студента с каскадной очисткой

#### Аналитика
- средний балл, прогресс, топ студентов

### 7. Theme и UX
- глобальная dark/light тема
- сохранение выбора в localStorage
- единый UI на всех страницах

### 8. Локализация
Поддерживаются 3 языка:
- казахский
- русский
- английский

### 9. Целостность данных
При удалении студента удаляются связанные сущности:
- `Project`
- `Rating`
- `Comment`
- связанные файлы

Это предотвращает появление orphan-данных.

### 10. Производительность и стабильность
- `async/await` + loading states
- корректная обработка ошибок
- build/lint проверки
- стабильный API-контракт

### 11. Короткий текст для защиты
> Тема моего дипломного проекта - веб-портал RoboLearn.
> Система автоматизирует учебный цикл: преподаватель публикует задание,
> студент отправляет проект, AI формирует первичную оценку,
> преподаватель выставляет финальный результат.
> Проект реализован на React + Node.js + MongoDB.
> Безопасность обеспечена JWT и ролевой моделью доступа.
> Внедрены мультиязычность, dark/light режим, загрузка файлов и аналитика.
> В результате получена стабильная и масштабируемая платформа,
> готовая к практическому применению.
